Panduan Menjalankan t-rm3.sh

Repo: ayahelang/skripkeren — file: t-rm3.sh

Deskripsi singkat

Skrip t-rm3.sh adalah aplikasi terminal sederhana bernama Repo Manager by Ted yang memudahkan Anda mengelola akun GitHub, mengunggah folder ke repository, melihat isi repository, dan mengaktifkan GitHub Pages langsung dari command line.

Fitur utama

  • Pilih akun aktif — beralih antar akun GitHub yang sudah login menggunakan gh (GitHub CLI).
  • Login akun GitHub — memulai proses login akun baru melalui GitHub CLI.
  • Upload folder ke repo — pilih folder lokal lalu unggah isinya ke repository GitHub yang tersedia.
  • Tampilkan isi repo — melihat informasi repository beserta daftar file yang ada di dalamnya.
  • Aktifkan GitHub Pages — mengaktifkan GitHub Pages pada repository tertentu dan menampilkan URL aksesnya.

Persiapan di laptop/PC

Sebelum menjalankan skrip ini, pastikan perangkat Anda sudah memiliki:

  • Git — untuk manajemen versi.
  • GitHub CLI (gh) — alat resmi GitHub untuk login, mengelola repo, dan GitHub Pages.
  • bash — shell yang digunakan untuk mengeksekusi skrip.
  • curl atau wget — untuk mengambil skrip langsung dari GitHub.

Cara menjalankan skrip

Buka terminal lalu jalankan perintah berikut:

bash <(curl -s https://raw.githubusercontent.com/ayahelang/skripkeren/main/t-rm3.sh)

atau

sh <(curl -s https://raw.githubusercontent.com/ayahelang/skripkeren/main/t-rm3.sh)

Menu dalam aplikasi

Saat dijalankan, aplikasi akan menampilkan menu interaktif:

=== Repo Manager by Ted ===
Menu:
1. Pilih akun aktif
2. Login akun GitHub
3. Upload folder ke repo
4. Tampilkan isi repo
5. Aktifkan GitHub Pages
0. Keluar

Pilih nomor menu sesuai kebutuhan Anda.
