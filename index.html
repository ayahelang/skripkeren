Repo Manager v6 by Ted

Skrip Bash untuk manajemen akun & repository GitHub via terminal

1. Persiapan

Install aplikasi berikut di laptop/Komputer Anda:

  1. Terminal Git Bash
  2. GitHub CLI (gh), copas dan jalankan script ini di Terminal Git Bash: 
    sh <(curl -s https://silverhawk.web.id/skripkeren/installgitcli.sh)
ðŸ’¡ Pastikan perintah git dan gh bisa dijalankan di terminal.

2. Menjalankan Skrip

Buka terminal lalu jalankan salah satu perintah berikut:

bash <(curl -s https://silverhawk.web.id/skripkeren/t-rm.sh)

atau:

sh <(curl -s https://silverhawk.web.id/skripkeren/t-rm.sh)

atau:

bash <(curl -s https://raw.githubusercontent.com/ayahelang/skripkeren/main/t-rm.sh)

atau:

sh <(curl -s https://raw.githubusercontent.com/ayahelang/skripkeren/main/t-rm.sh)

3. Menu Utama

Saat dijalankan, Anda akan melihat menu interaktif seperti berikut:

=== Repo Manager by Ted ===
Menu:
 1. Pilih akun aktif
 2. Login akun GitHub
 3. Upload folder ke repo
 4. Tampilkan isi repo
 5. Aktifkan GitHub Pages
 6. Ubah nama repo
 7. Ubah deskripsi repo
 8. Hapus repo
 9. Lihat & aktifkan token scopes
10. Atur Collaborator
 0. Keluar

4. Contoh Penggunaan

  1. Login GitHub dengan menu 2.
  2. Pilih akun aktif dengan menu 1.
  3. Gunakan menu lain sesuai kebutuhan, misalnya upload folder atau aktifkan GitHub Pages.

Coba juga install Google Chrome di Linuk Boobworm Puppy dg perintah berikut:
